# ES

## Cifrado Simple

#### Objetivo

Implementar un algoritmo de cifrado básico como César o XOR.

#### Qué aprenderás

- Conceptos básicos de criptografía.
- Manipulación de datos binarios y caracteres.

#### Extras

Implementar cifrados más avanzados como AES o SHA.

#### Extras
